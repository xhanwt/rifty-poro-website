Commands
We provide you the best, fast and reliable features in Bot!
Some commands have alias.
So, don't worry. Try typing in short form as you like and see if it works. ðŸ˜›
* Asterisk marked words are optional.
[] Bracked words are categories and user will have to write a custom word depending on the category.
r!help - Get help from poro.

r!prefix [prefix]* - Check guild's prefix. Set a custom prefix or reset to the default. (admin only)

r!ping - Pong!

r!ability [champion] [ability] - See info for a specific ability of a specific champion.
You can type passive or 0 and ability numbers (i.e., 1, 2, ...) in [ability].

r!build [champion] - Find the best item build of a specific champion from wildriftguides.gg.

r!so [champion] - Find the skill order of a specific champion from wildriftfire.com.

r!sr [champion] - Find summoners and runes of a specific champion from wildriftguides.gg.

r!stats [game mode] [tier]* - Find the champion stats of a specific game mode (and specific tier) from wildstats.gg.
Available game modes: ranked, rankedduo, pvp, aram

r!match - Find the upcoming and ongoing wild rift matches from liquidpedia.net.
